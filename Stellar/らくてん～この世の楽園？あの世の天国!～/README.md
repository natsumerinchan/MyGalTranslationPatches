## 使用方法
- 1.本补丁已集成特典补丁和免DVD补丁，无需另外下载
- 2.安装压缩包里的`unifont-all.ttf`字体(该字体是普通字体并非jis替换字体)
- 3.解压压缩包并把所有文件复制到游戏根目录下，双击运行rakutenDVD.exe
