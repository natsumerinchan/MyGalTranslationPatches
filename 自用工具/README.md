- 转换表`subs_cn_jp.json`来自[satan53x/SExtractor](https://github.com/satan53x/SExtractor.git)并根据我的个人使用习惯进行了[删减](https://gist.github.com/natsumerinchan/9a1f83f81e38f3e38683bdcc0ac5989f/revisions)
