## 使用方法
(请联合站内免DVD补丁使用)

解压压缩包并把所有文件复制到游戏根目录，双击SiglusEngine_CN.exe运行（如果有的话请删除游戏目录内的reg.ini，否则需要转区运行）

首次启动本补丁请删除savedata/config.sav，以便启动游戏后更改字体（游戏现已默认使用新宋体）
