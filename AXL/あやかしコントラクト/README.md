## 
- 1.安装压缩包里的`unifont-all.ttf`字体(该字体是普通字体并非jis替换字体)
- 2.解压压缩包并把所有文件复制到游戏根目录下，然后转区运行あやかしコントラクト.exe(否则语音会消失)
- 3.进入游戏后在`文字表示`-`フォント`中将字体改为`Unifont Smooth`
